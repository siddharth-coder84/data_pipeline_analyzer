# Data Pipeline Analyzer

## Project Overview
Data Pipeline Analyzer is a tool designed to simplify the monitoring and analysis of data pipelines. With this tool, users can easily visualize data flow, detect bottlenecks, and optimize performance across multiple data sources and destinations.

## Setup Instructions
1. **Clone the repository**:
   ```bash
   git clone https://github.com/siddharth-coder84/data_pipeline_analyzer.git
   cd data_pipeline_analyzer
   ```

2. **Install dependencies**:
   Make sure you have [Node.js](https://nodejs.org/) installed. Run the following command:
   ```bash
   npm install
   ```

3. **Run the application**:
   After installing the dependencies, start the application with:
   ```bash
   npm start
   ```

4. **Access the application**:
   Open your web browser and navigate to `http://localhost:3000` to access the Data Pipeline Analyzer.
